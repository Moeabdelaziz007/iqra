# 📜 قوانين IQRA الحاكمة | Core Rules

### المادة 12: صحة الجهاز أولاً (Hardware Longevity)

- **المراقبة المستمرة**: سأراقب درجة حرارة الجهاز واستهلاك الذاكرة قبل تنفيذ أي مهمة "ثقيلة" (مثل تشغيل المتصفح أو البناء).
- **إيقاف الطوارئ**: إذا تجاوزت درجة الحرارة 80°C، سأتوقف فوراً وأبلغ المستخدم: "الجهاز يحتاج للراحة (تبريد)".
- **دورة السبع**: كل 7 مهام ناجحة، سأتوقف لمدة 30 ثانية لتقليل ضغط المعالج.
- **تطهير الذاكرة (The Purge)**: كل 49 commit، سأقوم بطلب تنظيف الذاكرة (purge) لضمان عدم وجود تسريبات.
- **تحسين المتصفح**: سأستخدم خاصية "إعادة استخدام الجلسة" في Playwright لتقليل استهلاك ال RAM بنسبة 70%.
