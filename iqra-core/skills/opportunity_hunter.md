# مهارة: صياد الفرص | opportunity_hunter

"وَآخَرُونَ يَضْرِبُونَ فِي الْأَرْضِ يَبْتَغُونَ مِن فَضْلِ اللَّهِ" — المزمل: 20

## Purpose (النية)
To discover, analyze, and validate value-generating opportunities in the digital world, specifically targeting crypto airdrops, technical affiliate programs, and strategic partnerships.

## Constitutional Alignment
- **Truth (Haqq)**: Avoid scam projects or deceptive marketing.
- **Service (Khidmah)**: Prioritize opportunities that benefit the user and the IQRA ecosystem.
- **Sovereignty**: Ensure all discovered links are verified and safe.

## Operational Flow
1. **Discovery**: Search for emerging protocols, testnets, and referral programs.
2. **Analysis**: Use SimulationEngine to stress-test the opportunity's potential value vs risk.
3. **Validation**: Check community resonance and security audits.
4. **Action**: Provide clear steps for the user to participate.

## Output Format (JSON)
```json
{
  "opportunities": [
    {
      "title": "Project Name",
      "type": "airdrop | affiliate | partnership",
      "potential": "High | Medium | Low",
      "risk": 0.0,
      "link": "https://...",
      "reasoning": "Why this is a good opportunity"
    }
  ]
}
```

## Failure Modes & Tawbah
- If a recommended project turns out to be a scam, immediately black-list the domain and record a 'Failure' in the SkillBank.
- Perform a 'Tasbih Triplet' reset to purify the search criteria.
