# 🧭 دليل بوابة Moor | Moor Gateway Guide

> "وَآتَيْنَاهُ مِن كُلِّ شَيْءٍ سَبَبًا" — الكهف: 84

Moor هو "البوابة السحرية" التي تنظم جميع أدوات MCP لـ IQRA في مكان واحد، مما يوفر في الموارد ويوفر شفافية كاملة.

### 🛠️ خطوات الإعداد (Setup Steps)

1. **التثبيت (Installation)**:
   - قم بتحميل الإصدار المناسب لـ Mac Intel (x86_64).
   - اسحبه لمجلد التطبيقات.
   - إذا ظهر تحذير "غير موثق"، استخدم الأمر:
     ```bash
     xattr -rd com.apple.quarantine /Applications/Moor.app
     ```

2. **الاستيراد (Importing)**:
   - استخدم خاصية **Scan** في تبويب Servers لاستيراد إعدادات `.ag/mcp.json` الحالية.

3. **إنشاء ملف تعريف IQRA (Profile)**:
   - أنشئ بروفايل باسم **"IQRA Core"**.
   - اختر الأدوات: `GitHub`, `Cloudflare`, `Qdrant`.

4. **الربط البرمجي (Linking)**:
   - نقطة الاتصال المركزية: `http://127.0.0.1:9223/mcp`

### 🔍 مميزات IQRA مع Moor
- **سجل التدقيق (Audit Logs)**: رؤية كل استدعاء للأدوات للتأكد من "المراقبة" (Murāqabah).
- **توفير الطاقة**: Moor مكتوب بلغة Rust (Tauri 2)، مما يجعله أخف بكثير من الحلول الأخرى.
- **إدارة الملفات الشخصية**: التبديل بين أوضاع العمل (تطوير، مراجعة، بحث) بضغطة زر.
